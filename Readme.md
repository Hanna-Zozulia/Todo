# Todo

---

## Описание

Todo — это компактное Kanban-приложение, написанное на чистом TypeScript, работающее прямо в браузере без сервера. Все данные сохраняются в `localStorage`, что позволяет не терять задачи после перезагрузки страницы.  

Пользователь может:

- Создавать задачи через форму с указанием приоритета (`Normal` / `Urgent`).  
- Перетаскивать задачи между колонками с помощью Drag & Drop.  
- Отмечать задачи как выполненные, с автоматической записью времени завершения.  
- Сохранять и отображать все задачи в реальном времени на доске.  
- Автоматически видеть метки времени:  
  - `createdAt` — дата создания  
  - `finishedAt` — дата выполнения  

Приложение разработано с использованием TypeScript, HTML и CSS и демонстрирует навыки работы с DOM, событиями, drag-and-drop и управлением состоянием приложения.
  
### Колонки на доске

- **To Do** — задачи, которые нужно выполнить  
- **In Progress** — задачи в работе  
- **Testing** — задачи на проверке  
- **Done** — завершённые задачи

## Установка и запуск

1. Клонируйте репозиторий:

```bash
git clone 
```

2. Установите зависимости:

```bash
npm install
```

4. Запуск проекта:

Проект не требует backend-сервера — он полностью статический.
Для удобного запуска можно использовать расширение **Live Server** в VS Code.

- Запуск через Open with Live Server.
- Установите расширение Live Server в VS Code.
- Откройте проект в VS Code.
- Кликните правой кнопкой по файлу index.html.
- Выберите Open with Live Server.

Браузер автоматически откроет страницу по локальному адресу, например:

```bash
http://127.0.0.1:5500/
```

Live Server автоматически обновляет страницу при изменении файлов, что удобно во время разработки.
